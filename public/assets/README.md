# Assets Directory

Place your images and other assets here:

- `avatar.jpg` - Your profile photo (150x150px recommended)
- `project1.jpg` - Portfolio project 1 screenshot (800x500px recommended)
- `project2.jpg` - Portfolio project 2 screenshot
- `project3.jpg` - Portfolio project 3 screenshot
- Add any other images, icons, or assets you need

## Image Optimization Tips

1. Use WebP format for better compression
2. Optimize images before uploading (use tools like TinyPNG)
3. Recommended dimensions:
   - Profile avatar: 300x300px
   - Project screenshots: 800x500px or 16:9 ratio
   - Icons: SVG format when possible

## Placeholder Images

The app will automatically use placeholder images if your custom images aren't found. Replace them with your actual images before deployment.
